ReadMe.txt

First need to clone repositary https://github.com/Slaporter/comp30670-assignment3.git
This can be installed using pip (pip install git+https://github.com/Slaporter/comp30670-assignment3.git)

Ensure tqdm is installed (pip install tdqm)
Once these are installed, from the command line run the application using the urls in the style:
1. solve_led --input http://claritytrec.ucd.ie/~alawlor/comp30670/input_assign3.txt
2. solve_led --input http://claritytrec.ucd.ie/~alawlor/comp30670/input_assign3_a.txt
3. solve_led --input http://claritytrec.ucd.ie/~alawlor/comp30670/input_assign3_b.txt
4. solve_led --input http://claritytrec.ucd.ie/~alawlor/comp30670/input_assign3_c.txt


A status bar will show showing the progress of the program.
